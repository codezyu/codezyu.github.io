# 当官
大部分考公务员，除了享受平淡安稳的生活，同样有着仕途的向往，向往的是生活有管家，上班有跟班，平时只需要讲稿子，拍板子即可。但是这可能吗？ 。
我们追求的是官，但是考公给的是吏。吏是下面跑腿的，官才是上述的享受者。因此核心目标一定是去当官。
## 优选 技术岗转官员
第一份工作，最好选择高等院校，体制内的科研院所，媒体，重要的国有企业。这条路的门槛是需要博士学位，并且是名校
通过公务员考试，走到 **中央部委的技术岗位**。
原因在于他们允许通过技术线走到非常高的岗位，并且用人自由度非常高，比党政机关高，非常适合奋斗流。
最后需要通过伯乐把你**调任**到党政机关，这一步非常吃人脉，比如说**清华大学**，在副厅级最好开始调任。县委书记是一个坑，如果不为财，最好绕过。晋升到省部级非常吃机遇。

一是因为党政机关的晋升难度非常高，只有是地方主官才有相对可靠的晋升机会，同时一路上上去都得关系，因为上去的每一个职位都是肥差和实务岗位。
二是选官条例里面增加了对上述岗位的偏好， **可以调入机关担任领导职务或者四级调研员以上**
三是当调任党政机关前，你已经有了很好的技术成绩，背景很好，官员成绩要求就不会要求太高。中年出任当官，背景同时很清白，不容易卷入到派系中。
四是绕过了基层工作，绕过了微薄的收入，从吏到官的门槛。
五是目标工作岗位是大城市，生活待遇比乡村好太多，同时哪怕失败了，也是大城市的高级技术员
## 次选：共青体系
通过高校的行政体系进入本省市的共青团系统，大部分985的校团委是正处级，和县委书记同级
一相比进入基层的县委书记，更轻松
但是这条路也不是都是好的
一是没有实权，哪怕得到了交流学习机会，没有班底，也没有同僚，**官场根基浅薄**是大忌
二是上面讨厌共青团派系，认为夸夸其谈，存在歧视，这是最大难点
三是党八股研究对实际工作无用处，因此现有的共青团干部外发到党政机关只能得到降职重用。共青团虽然升得快，但是领导班子对年龄限制基本没有了
相比基层，有地利，没有天时，人利
## 弯路：选调
一选调生之间差异极大，只有中央选调和厅级以上单位的选调生值得考虑，但是最年轻的厅级干部都是来自共青团和企事业单位调入。大部分选调生是培养乡镇干部的摇篮，天花板是贫困县的县委书记
二艰苦环境，低保薪水，远离家人

# 官员背景知识
## 官员等级
分正副之外，然后为
- 国
	- 正国级：只有7个，习近平54岁，胡锦涛<50岁，江泽民63岁。江泽民在副国级临危受命，习近平培养5年后，胡锦涛10年后
	- 副国级：中央政治局委员，上海市市长，基本没有更高的职级了，由一群人进行选举，主要是选举候选人。19大采用谈话调研，面对面听取，不限定推荐人数，弱化选票推荐。
- 部
	- 正部：最快两人，省长，部长等。中央政治局表决（40人）习近平46岁，胡锦涛42岁，江泽民57岁
	- 副部：总共2800人，省会的市长，由中央政治局常委（7人）表决，省委有推荐，习近平40岁，胡锦涛40岁
- 厅
	- 正厅：市长，市委书记，科员到这至少需要25年，习近平33岁，胡锦涛40岁。从副厅需要至少两年，省委人任免权
	- 副厅：副市长，公安副厅长，本科学历要求。从处到厅理论最快需要三年，习近平32岁，胡锦涛38岁。
- 处
	- 正处：县处级 10% 5年工龄，两年基层工作经验，博士生默认有三年工作经验，一般要有下一级两个以上的职位经历（比如当过乡长和乡党委），每年需申报个人情况（和境外的联系等，财产等），古代七品到这。从副处升上去至少两年，任免权在省委。
		- 市委书记对县处级有建议罢免权。
		- 除了县委书记，其它县处级由市委管。
		- 县委书记权力很大，习近平30岁，胡锦涛跳过该阶段
	- 副处：正科到副处，最快需要三年，胡锦涛33岁到这，习近平29岁。如果45岁还没升迁，以后大概也得不到了。
- 科
	- 正科：乡委党书记 任命受到上级直接控制
	- 副科：乡长
越级提拔一般效果不好，而且限制严格，容易被社会关注，基本只能越一级。限制办法有追责问题，对提拔的人有追责
- 家世背景
- 学历
- 个人能力
一般至少两个条件，还得加上运气。
职级还得考虑实权，比如政协同级权力就很小。
选拔流程
- 动议：领导提议
- 推荐：相关领导推荐（包括退休）
- 考察：政绩考核
- 决议：上级表决
- 履职：公示，培训

中国公务员在16年就有716万人，公务员最高只有正厅级
国考：国家公务员考试，录取率比较低，2025为341.6 录取仅为3万人左右
选调生：直接从高校选拔后备资格，学校越好，越不挑专业，提拔速度比普通公务员快。
厅级选拔需要被公布，学历会成为对手攻击重要点

# 国企
国企大部分效益不好，容易受地方财政影响，受到货款拖欠等等。
- 基建，煤炭，钢铁这些问题最严重
- 公共事业的国企利润一向微薄，比如水电，燃气等等，经济不好的时候更容易出问题
## 企业分类
- 国家的核心企业，电网，烟草，三桶油，通信等属于第一梯队，必有国家支持，基本不受周期影响
- 战略发展，航天，受益于重点研发项目
- 保障基本民生，政权文档，比如粮食安全
- 新生产业，比如新能源，中车的轨道交通，关键在于国家的态度。
