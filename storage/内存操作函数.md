在热点图中，如果对string操作比较多，很容易看到有memset函数，不过我们经常看到的是
```cpp
__memset_sse2 // 使用 SSE2 指令集优化
__memcpy_ssse3_back // 使用 SSE3 
```
在SIMD中，可以实现一次操作多个字节
- SSE2: 128位/16个字节操作 基础的向量加载/存储
- SSSE3 : 更复杂的指令操作，比如字节重排指令
为什么memset和memcpy使用不同的指令，是因为memcpy需要考虑对齐，覆盖问题
## memset
在`__memset_sse2`中，可以认为操作为
```cpp
__m128i pattern = _mm_set1_epi8(value); // 包含16个相同value的字节
_mm_store_si128((__m128i*)(p), pattern); // 存入
```
