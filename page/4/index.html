python序列类型

Posted on 2021-01-24 Edited on 2021-01-25 In 编程基础

python序列类型

1.列表

创建:[]

访问：正向，负向，从0开始

​ 切片访问：左边边界元素不访问

删除:del

​ pop(根据索引值删除)

​ remove(根据元素值删除)

​ clear(删除所有元素)

添加元素:+,append(整体加入),extend(逐个加入)

插入：insert(剩余元素后移)

修改：直接赋值

​ 用[]对切片进行赋值(空切片相当于插入)

查找：index查找元素的索引值

​ count统计元素出现次数

函数：range(for value in range(begin,end):)不包括右边界

2.元组

相比序列，元组可看作不可修改的序列（修改，插入，删除），但元组可以覆盖

创建:()
访问:[i]

删除：del

3.字典

键值对，无序，可嵌套（类似广义表）,键值唯一（否则保留最近的键值对），键不可变（数字，字符串，元组）

创建：{key：value}

访问：[key]

       get(key)

删除：del

添加：[new key]=new value

修改：只能修改value，不能修改key

​ a[key]=new value

删除:del [key]

查找：判断key是否存在 in/not in

4.序列

集合，保存不重复元素(数学好像是允许集合重复的),存储的数据类型不可变，元素无序

创建：{}

访问：一般用循环读取想要元素

删除：del

​ remove(element)

添加：add()

运算:交集& 并集| 差集- 对称差^

清空：clear()
