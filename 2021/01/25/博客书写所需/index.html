hexo使用

1.命令

hexo clean 对当前网站内容进行清理

每次更新推文需要输入命令：hexo clean && hexo g && hexo d

hexo g 生成

hexo d 部署

上面两个可以合并为 hexo d -g

hexo s本地预览

hexo n+”name” 生成为name的md文档，可以在这里直接写博客
